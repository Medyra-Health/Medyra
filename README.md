# Medyra

AI-powered platform that helps patients understand their medical lab results in plain language.

**Live:** [medyra.de](https://medyra.de)

---

## Setup

**Prerequisites:** Node.js 18+, Yarn

```bash
git clone https://github.com/Medyra-Health/Medyra.git
cd Medyra
yarn install
cp .env.example .env.local
# Fill in .env.local with your credentials
yarn dev
```

App runs at `http://localhost:3000`.

## Environment Variables

See `.env.example` for all required variables.
